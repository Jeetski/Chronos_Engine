# Familiar Files

Core data files
- `meta.json`: identity and UI traits.
- `state.json`: live emotion, avatar, location, hearts.
- `profile.json`: user profile and consent flags.
- `memory.json`: opt-in memory store.
- `skills.yml`: optional model/tool prefs.

Persona docs live in `familiars/<name>/docs/`. See `directory-structure.md`
for the full list.
